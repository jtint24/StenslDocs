# StenslDocs

This repo contains the documentation for the Stensl Language, currently in version a1.0.0. Stensl is a new scripting language that runs in the JVM. Currently, itâ€™s in alpha version; it has baseline features but lacks some capabilities and stability guarantees of a public language. That being said, Stensl has a variety of powerful and unique features. Stensl is a gateway to powerful and diverse software; see what you can build with it!
